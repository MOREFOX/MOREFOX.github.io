# MOREFOX.net Ver 2.6.31 版本发布！！！

链接：网页链接

-修复bug：刷新界面信息框背景不加载问题。

发现一个奇特的bug，这个bug导致刷新界面时，站点概览板块的背景时透明的，极度影响美观，进行了长达四天的寻找bug之旅，最终修复成功，方法为关闭皮肤config内透明过度动画即可。

-界面优化：修改了next主题，加载速度增快。

next主题从Muse主题替换为Gemini主题，经测试Gemini主题加载速度更为迅速和流畅。

-界面优化：修复背景与主页文章同层问题。

不是很喜欢next主题集成的动态背景，测试很久背景的替换，都不约而同地出现了背景和博文板块同层的问题，经过和Cherry’s Blog 内代码的对比，最终修复了这个问题。

-评论系统优化：删除Disqus评论系统，增加Gitment评论系统。

Disqus评论系统在未科学上网的情况下时无法加载和显示的。最终决定使用gitment评论系统。

-优化：删除网页屏幕中间加载条，改为右上角百分比。

由于网页代码还存在纰漏，网页加载经常卡在99%，但在99%时却可以正常操作，屏中加载条十分挡视线，所以替换为右上角百分比，提升观感和操作体验。

-代码托管服务器优化：迁移至coding托管服务器。

已全面删除github代码托管文件，迁移至coding托管服务器，加载速度至少缩短300%。 （但年租金挺贵
